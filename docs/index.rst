Welcome to pyGLV's documentation!
==================================

pyGLV stands for computer **G**\ raphics for deep **L**\ earning and scientific **V**\ isualization in **py**\ thon!

A python, pure software design pattern based package that 
used plain and simple Entities, Components and Systems in a 
Scenegraph architecture from the ``pyECSS`` package, in order to showcase basic, 
cross-platform OpenGL-based real-time computer graphics with applications to 
scientific visualization and deep learning.
